# Desktop Version

### Install Plugin

1.  Navigate to **`Extensions`** and search for _**IZ scan**_ and click on **`Install`** <br>

    <figure><img src="../../../../../../.gitbook/assets/search-iz-scan.png" alt=""><figcaption></figcaption></figure>
2.  **`IZ Scan`** extension can also be installed from the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=integralzone.iz-scan). Clicking on **`Install`** button will open the extension installation page in VS Code. <br>

    <figure><img src="../../../../../../.gitbook/assets/iz-scan-marketplace.png" alt=""><figcaption></figcaption></figure>
3. Restart VS Code after installation
