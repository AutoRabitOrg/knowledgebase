# Configure Category

### Basic Details

1. Navigate to **`IZ Pulse`** -> **`Categories`** and click on **`Create Category`**
2. Enter a name for the category and click on Submit

### Add Endpoints

1.  Search for a **`Endpoint`** and select the same to add it as part of the category <br>

    <figure><img src="../../../../.gitbook/assets/add-category-endpoint.png" alt=""><figcaption></figcaption></figure>
2.  The selected endpoints will be displayed in a table<br>
