# Scheduling Scans

## Scheduling Scans For External Repositories

### Sync Repositories from External Sources - **`IZ Scan Sync`** Job

1. Navigate to **`Schedules`** -> **`Configure Schedule`**
2. Choose **`IZ Scan Sync`** job type. This job will sync the repositories from the selected external sources.
3. Select the external source and the organization
4. External Sources can either be -
   1. Bitbucket
   2. GitHub
   3.  Anypoint Design Center<br>

       <figure><img src="../../../../../.gitbook/assets/image (2570).png" alt=""><figcaption></figcaption></figure>
5. Select the required schedule. Schedule can either be -
   1. One Time
   2.  Recurring<br>

       <figure><img src="../../../../../.gitbook/assets/image (2572).png" alt=""><figcaption></figcaption></figure>

### Configure Scan Schedules - **`IZ Scan`** Job

1. Navigate to **`Schedules`** -> **`Configure Schedule`**
2. Choose **`IZ Scan`** job type. This job will scan the selected repositories.
3. Select the external source and the organization
4. External Sources can either be -
   1. Bitbucket
   2. GitHub
   3.  Anypoint Design Center<br>

       <figure><img src="../../../../../.gitbook/assets/image (2570).png" alt=""><figcaption></figcaption></figure>
5. Select the required schedule. Schedule can either be -
   1. One Time
