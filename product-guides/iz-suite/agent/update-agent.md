# Update Agent

Agents and workers are responsible for executing the schedules

{% hint style="warning" %}
* Agents require JRE 11 or greater to be installed
* Agent type cannot be updated once the agent is created
{% endhint %}

1. Navigate to **`Global Settings`** -> **`Agents`**&#x20;
2. Click on **`Edit`** action against the agent and update the values
