# Notifications

## Notifications

### Subscription

Users can subscribe to the events listed below through any of the supported channels, such as Email, Slack, or Web.

### License Notification Events

Notifications related to License Modules and Validity

| Event                     | Description                                                               |
| ------------------------- | ------------------------------------------------------------------------- |
| On License Expired        | Event which will be triggered on IZ Suite license expiry.                 |
| On License Quota Exceeded | Event which will be triggered when any of the allotted quota is exceeded. |

### Organization Notification Events

Notifications related to Organization

| Event                                     | Description                                                                                |
| ----------------------------------------- | ------------------------------------------------------------------------------------------ |
| IZ Eye - On Org On Board                  | Event which will be triggered when a new Organization or Business Group is on boarded.     |
| IZ Eye / Scan - On Mule App Scan          | Event which will be triggered when a Mule application is scanned.                          |
| IZ Eye / Scan - On Mule Quality Gate Fail | Event which will be triggered when Quality Gate of a Mule application being scanned fails. |
| IZ Eye / Scan - On API App Scan           | Event which will be triggered when an API application is scanned.                          |
| IZ Eye / Scan - On API Quality Gate Fail  | Event which will be triggered when Quality Gate of an API application being scanned fails. |
