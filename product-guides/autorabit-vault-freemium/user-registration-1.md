# User Registration

**AutoRABIT Vault Freemium User Registration**

User Guide

This guide explains the Freemium registration workflow, including account sign-up, agreement acceptance, OTP verification, and the confirmation email received after the free trial account is activated.

| **Workflow outcome:** After registration and verification, the account is activated for the AutoRABIT Vault free trial. The activation email provides access details, the instance URL, the master code, and the applicable trial limits. |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

## Purpose

The Freemium registration flow allows a new account to be requested and activated for a limited AutoRABIT Vault free trial. The workflow captures basic account details, requires agreement acknowledgement, verifies the registration using OTP, and sends an activation email with the trial access information.

{% hint style="info" %}
**Note:** Vault Free Trial access remains active for **14 days** from the subscription start date. After the trial period ends, account data is retained for an additional **7 days**. Once this retention period is complete, the account is permanently deleted, and no user data is retained after deletion.
{% endhint %}

## Registration Workflow Overview

| **Step**                         | **Description**                                                                                                |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Access the registration page** | The AutoRABIT Vault Freemium sign-in page provides access to the registration option for new accounts.         |
| **Enter registration details**   | The registration form captures the required personal, organization, and regional details.                      |
| **Accept the agreement**         | The agreement dialog is reviewed and acknowledged before registration proceeds.                                |
| **Complete OTP verification**    | The OTP email is used to complete the verification step.                                                       |
| **Receive activation details**   | The activation email confirms that the free trial account is active and lists access details and trial limits. |

## Starting the Registration Flow

The registration journey begins from the AutoRABIT Vault Freemium sign-in page. Existing accounts can sign in using the registered email address. New accounts proceed through the sign-up option displayed on the same page.

## Entering Registration Details

The registration form captures the required account information. The form includes fields for the registrant name, account email, organization, and region or location. The selected region determines the free trial instance location used for the account.

Required fields must be completed before the registration can continue. The regional selection is made from the available options in the Region/Location list.

## Activation Email and Trial Information

After verification and activation, AutoRABIT Vault sends the free trial activation email. The email confirms that the registration is complete and that the account is active. It also directs the account holder to add a Source Org from the dashboard to begin configuring backup and other AutoRABIT Vault activities.

### Access Details Included in the Activation Email

The activation email includes an access details section. The section provides the subscription start date, trial period, trial expiry date, registered email address, storage location, instance URL, and master code. These details are used to access and manage the free trial instance.

| **Email field**             | **Purpose**                                                                |
| --------------------------- | -------------------------------------------------------------------------- |
| **Subscription Start Date** | Shows when the free trial subscription begins.                             |
| **Trial Period**            | Shows the length of active free trial access.                              |
| **Trial Expiry Date**       | Shows when the trial access ends.                                          |
| **Email**                   | Shows the registered account email.                                        |
| **Storage Location**        | Shows the region assigned to the trial instance.                           |
| **Instance URL**            | Provides the URL used to access the AutoRABIT Vault instance.              |
| **Master Code**             | Provides the code associated with account access and verification support. |

### Free Trial Limits

The following free trial limits are communicated in the activation email. These limits are listed once here to avoid repeating the same details across the guide.

| **Limit**                         | **Allowed value** |
| --------------------------------- | ----------------- |
| **Storage**                       | 2GB               |
| **Files**                         | 10GB              |
| **Users**                         | 1 Admin           |
| **Environments**                  | 2 Salesforce Orgs |
| **Backup Configs**                | 3                 |
| **Backup Jobs**                   | Unlimited         |
| **Restore Jobs**                  | 3                 |
| **Replicate / Data Seeding Jobs** | 3                 |
| **Retention Period**              | 14 days           |

AutoRABIT Vault usage and trial limits can be monitored from the AutoRABIT Vault dashboard after the account becomes active.
