# Codescan API overview

The CodeScan API Documentation provides a centralized reference for all available APIs to interact with the CodeScan platform. It outlines endpoints, request/response formats, parameters, and access requirements to help users integrate, automate, and manage workflows efficiently.

Most of the API documentation for the CodeScan Cloud platform can be found on the platform under the **Help Menu (**<img src="https://cdn-icons-png.flaticon.com/512/9603/9603852.png" alt="" data-size="line">**) > Web API**.

Platform specific API documentation (such as Audit Logging) can be found here on the Knowledge Base.
