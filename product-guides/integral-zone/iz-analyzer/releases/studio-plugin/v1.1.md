## Release - v1.1

Release Date - 10 Feb 2021

Studio Plugin feature version - 1.0.0.202102091632

Release notes - IZ Analyzer - Version IZ Analyzer Studio Plugin - v1.1

* Feature
  * \[MSQA-187] Enable Quality Profile Selection - Anypoint Studio

In previous versions, rules to scan the project would be downloaded from the Quality Profile marked as default. There was no option to choose the required Quality Profile from Preferences.

This release enables users to choose the required Quality Profile in `IZ Preferences`. Refer to [documentation](../../../iz-suite/iz-scan/anypoint-studio/install-iz-analyzer-studio.md) for more information.
