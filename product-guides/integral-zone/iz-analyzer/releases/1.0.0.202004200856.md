# 1.0.0.202004200856

## Release - 1.0.0.202004200856

### Release Notes:

1. Scanning of projects will not block the UI any more.
2. Introduced a new feature to Start and Stop Auto Analysis. The Stop/Start button has been included in the IZ Mule On The Fly Results pane as shown below.
   1.  Project auto scan can be stopped at any point of time using the stop button below. <br>

       <figure><img src="../../../../.gitbook/assets/1.0.0.202004200856-on-the-fly-stop.png" alt=""><figcaption></figcaption></figure>
   2.  Project auto scan can be started at any point of time using the start button below. <br>

       <figure><img src="../../../../.gitbook/assets/1.0.0.202004200856-on-the-fly-start.png" alt=""><figcaption></figcaption></figure>
3. Project scanning will be done in a background schedule job. If a project scan is already scheduled, any subsequent scan requests (basically on saving the file) will be ignored.
