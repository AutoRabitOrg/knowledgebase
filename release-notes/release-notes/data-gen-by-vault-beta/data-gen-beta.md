# Data Gen - Beta

Data Gen introduces intelligent synthetic test data generation for Salesforce environments.

#### Key Capabilities <a href="#key-capabilities" id="key-capabilities"></a>

* **Salesforce Schema Analysis:** Connects to a Salesforce org and analyzes its object schema, fields, and relationships.
* **LLM-Based Data Generation:** Generates contextually relevant synthetic data based on the analyzed Salesforce schema.
* **DataFaker-Based Generation:** Creates realistic synthetic data using DataFaker.
* **Agentic Flow Data Generation:** Reads Salesforce configurations and generates synthetic data required to test Salesforce agentic flows.

> Data Gen is intended for test data generation. Connecting a production Salesforce org is not recommended.
