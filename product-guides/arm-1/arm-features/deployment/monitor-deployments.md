# Track Profile & Permissions Compare History

{% hint style="info" %}
The **Profile & Permissions Compare History** screen is best viewed when the zoom setting is set to **80%** on your Chrome/Firefox browser.
{% endhint %}

### Overview <a href="#overview" id="overview"></a>

\
The Profile Compare History page provides a centralized view of all profile and permission set comparison activities performed within ARM. This page helps administrators track what comparisons were created, between which Salesforce orgs, and by whom, making it easier to audit changes and manage historical comparisons.\
<br>

<figure><img src="../../../../.gitbook/assets/Screenshot 2026-02-04 at 10.33.41 AM.png" alt=""><figcaption></figcaption></figure>

**What you can do on this page**\
\
• View all profile and permission set comparison records created in ARM\
• Identify the source and target Salesforce orgs used for comparison\
• Track who created each comparison and when\
• Search, filter, and navigate through historical comparison entries\
• Delete comparison records that are no longer required
