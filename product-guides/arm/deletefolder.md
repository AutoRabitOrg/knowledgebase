# DeleteFolder

To create this template, follow the below steps:

1. Login to your [AutoRABIT](https://www.autorabit.com/) account.
2. Click on **Env. Pro.** module.
3. Click on **Create New Template**.
4. Go to the **Create Unsupported Metadata Template** tab.
5. Give the template a **name** and a **short description** of it.
6. Select the **DeleteFolder** checkbox available under **Report Dashboards Create Manage Folders**.
7. Click **Add**.
8. On the next screen, a **Test Case Name** appears automatically by default. To add the custom test data, click on **Add** button and give a name to them.&#x20;
9. In the **Delete Folder Name** field, type the folder name you want to delete. To save this page, click the **Save** button.
10. Next, Click **Save** to save the template.
11. Once the template is successfully created, you'll be redirected to the **Environment Provisioning History** screen.
12. Click the **Run** button to run the template on your destination org.
13. Select your **destination org** from the dropdown and enter the **email address(es)** to receive an email notification whenever the template is run.
14. In the **Post Deployment Steps**, select the test cases that you have recently created.&#x20;
15. Please check the **View History** page for a detailed summary report of the operation carried out.
