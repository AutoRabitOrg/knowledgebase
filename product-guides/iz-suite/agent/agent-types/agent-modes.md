# Agent Modes

**`Agent`** can be started in 2 different modes -

### Cloud Hosted

Cloud hosted agents can be used when you do not want -

1. To install the pre-requisite softwares required by the agent (i.e. Java)
2. To run the agent on your own machines or if you do not have any servers
3. The Agent / Workers to download and scan applications on cloud

### Self Hosted

Self hosted agents can be used when you do not want -

1. To share the Anypoint Connected App’s Client Id and Secret
2. The Agent / Workers to download and scan applications on cloud
