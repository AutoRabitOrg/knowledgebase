# Configure Schedule

{% hint style="info" %}
* Repeatedly creating the same schedule will just overwrite the pre-existing schedule.
{% endhint %}

1. Navigate to **`Schedules`** -> **`Schedules`** and click on **`Configure Schedule`**
2.  Select **`Health Check`** job type\
    &#x20;

    <figure><img src="../../../../../.gitbook/assets/health-check-job-schedule1.png" alt=""><figcaption></figcaption></figure>
3.  Select the schedule/frequency at which the analysis should be performed<br>

    <figure><img src="../../../../../.gitbook/assets/health-check-job-schedule2.png" alt=""><figcaption></figcaption></figure>
4. Click on **`Submit`** to configure the schedule
