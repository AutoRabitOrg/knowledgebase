# Source Code Analysis

## About Source Code Analysis

IZ Analyzer helps developers detect issues at the time of development, i.e., even before the issue exists.

Some of the key benefits are:

* Write better quality code
* Less turnaround time to fix quality-related issues
* Instant detection of quality violations during development
* Recommendations on issue fix

### In Anypoint Studio

Studio plugin helps developers detect and fix quality issues at the time connectors are added to the flows. In case any issues, precise details of issue, recommendation on issue fix, line number will be available to the developer.

### In Server

Once the development is complete, projects can be uploaded to the Server for maintaining the history using a **`sonar-scanner`** or **`maven`** option.
