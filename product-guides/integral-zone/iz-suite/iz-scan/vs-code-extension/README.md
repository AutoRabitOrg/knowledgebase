# VS Code Extension

## IZ Scan - VS Code Extension

### Overview

IZ Scan is a powerful code analysis tool for VS Code, empowering developers to ensure high-quality code by leveraging custom and pre-built rule sets. Developed by Integral Zone, it’s designed to work seamlessly with various projects, including Mule and API. This extension helps developers and teams ensure code quality and compliance with best practices in their MuleSoft and API development.

### Features

1. **`Static Code Analysis`**: Scan your Mule projects and API projects (RAML, OAS) for potential issues, code smells, and security vulnerabilities.
2. **`Real-time Feedback`**: Receive real-time feedback and recommendations as you code, helping you catch and rectify problems early.
3. **`Customizable Rules`**: Configure rules and checks to match your project’s specific requirements and coding standards.
4. **`Project Integration`**: Seamlessly integrates with your Mule projects and API projects, providing insights within your development environment.
5. **`Pre-built Rules`**: Out of the box, IZ Scan includes 160+ Mule-specific rules and 50+ API-specific rules.
6. **`Rule Profiles`**: Organize rules into profiles for targeted code evaluations.
7. **`Auto-fix Capability`**: Suggest and apply fixes to code issues automatically.
8. **`Integrated Issue Viewer`**: View all identified issues directly within your project files in VS Code.
9. **`Cross-Platform Support`**: Works on Windows, macOS, and Linux.
