# IDE Settings

{% hint style="warning" %}
* This feature is available only in Anypoint Studio Plugin version 5.1.3 and Anypoint Code builder version 1.0.8.
* Any changes to the settings mentioned will be synced by **`Anypoint Studio Plugin`** periodically. For changes to take effect immediately, developers have to sync the metadata manually.
{% endhint %}

### IDE Settings

Configuration to control the projects that are being scanned from IDE.

1. Navigate to **`Settings`** -> **`Global Settings`**.
2. Search for **`IDE Settings`** and click on edit.
3. Updated the required pattern to exclude or include the projects -
   1. **`Project Name Include Pattern`** - By default, all the projects will be scanned
   2. **`Project Name Exclude Pattern`** - By default, no projects will be excluded from scan
   3. **`Blacklisted Projects Message`** - Message to be displayed when scan is not allowed based on the pattern configuration
